# homebrew-tap

## Installation

Add the tap:

```shell
brew tap thegeeklab/tap
```

Install formula:

```shell
brew install <formula> # e.g. brew install git-sv
```

## Update formulas

```shell
# make sure you installed the formula first, then:
brew bump-formula-pr -v --strict --version <new version> git-sv
```
